# مدیریت سریع - نسخه سفارشات و رنگ محلی

این نسخه شامل تغییرات زیر است:

- رنگ کارت محصولات دیگر وابسته به برگشت اطلاعات از سرور نیست؛ خود اپ با توجه به دکمه‌ای که زده شده رنگ را ثبت می‌کند.
- اگر محصول بروزرسانی شود: کارت تا ۲۴ ساعت آبی می‌شود.
- اگر محصول به بله ارسال شود: کارت تا ۲۴ ساعت سبز می‌شود.
- اگر هر دو کار انجام شود: کارت تا ۲۴ ساعت گرادیانت سبز/آبی می‌شود.
- بعد از زدن «ذخیره تغییرات» یا «ارسال در کانال»، صفحه محصول بسته نمی‌شود.
- برای خروج از صفحه جزئیات، دکمه ضربدر بالای صفحه را بزنید.
- بخش «سفارشات» اضافه شد.
- لیست سفارش‌ها با زمان نسبی مثل «چند لحظه قبل»، «۱۰ دقیقه قبل»، «۲ ساعت قبل» نمایش داده می‌شود.
- جزئیات سفارش شامل محصولات، تصاویر، قیمت‌ها، مبلغ کل، درگاه پرداخت، آدرس، شماره تماس و کدپستی است.
- امکان ساخت و اشتراک فاکتور سفارش به صورت PNG اضافه شد.

## نصب افزونه وردپرس

داخل پوشه `wordpress-plugin` افزونه را zip کنید یا از فایل zip آماده استفاده کنید و در وردپرس نصب کنید:

`افزونه‌ها → افزودن → بارگذاری افزونه`

بعد افزونه را فعال کنید.

## ساخت APK با GitHub

اگر پوشه مخفی `.github` در ویندوز آپلود نشد، فایل `GITHUB_ACTION_CODE.txt` را باز کنید و محتوای آن را در مسیر زیر در GitHub بسازید:

`.github/workflows/build-apk.yml`

بعد از Commit، از تب Actions خروجی APK را از بخش Artifacts دانلود کنید.

## کرون‌جاب

کرون جدید لازم نیست. همان کرون قبلی کافی است:

```bash
wget -q -O - "https://banommi.com/wp-cron.php?doing_wp_cron" >/dev/null 2>&1
```
