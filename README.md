# مدیریت بانومی - اپ مدیریت محصولات ووکامرس

این پروژه یک monorepo ساده است:

```text
mobile/              اپ Flutter اندروید
wordpress-plugin/    افزونه وردپرس برای REST API امن
.github/workflows/   اکشن ساخت APK در GitHub
```

## امکانات نسخه جدید

- نام اپ: **مدیریت بانومی**
- سازنده: **شهرام سعیدنیا**
- ورود با نام کاربری و رمز وردپرس
- ذخیره امن آدرس سایت، نام کاربری، رمز و توکن روی گوشی
- پر شدن خودکار فیلدهای ورود بعد از اولین ورود
- ورود سریع با اثر انگشت / قفل امن گوشی
- نمایش دسته‌بندی‌های محصول
- نمایش محصولات هر دسته‌بندی
- نمایش تصویر محصول در لیست محصولات
- جستجوی محصول داخل همان دسته‌بندی
- صفحه جزئیات محصول
- ویرایش قیمت، تعداد موجودی و وضعیت موجود / ناموجود

> نکته امنیتی: اطلاعات ورود با `flutter_secure_storage` در حافظه امن گوشی ذخیره می‌شود. ورود اثر انگشتی با پکیج `local_auth` انجام می‌شود.

## نصب افزونه وردپرس

1. وارد پوشه زیر شو:

```text
wordpress-plugin/woo-mobile-stock-manager-api
```

2. همین پوشه `woo-mobile-stock-manager-api` را zip کن.

3. برو پیشخوان وردپرس:

```text
افزونه‌ها → افزودن → بارگذاری افزونه
```

4. فایل zip افزونه را آپلود و فعال کن.

5. ووکامرس باید نصب و فعال باشد.

## سطح دسترسی کاربر

کاربری که داخل اپ لاگین می‌کند باید یکی از این دسترسی‌ها را داشته باشد:

```text
manage_woocommerce
edit_products
```

معمولاً مدیر سایت یا مدیر فروشگاه این دسترسی را دارد.

## اجرای Flutter روی سیستم

داخل پوشه پروژه:

```bash
cd mobile
flutter pub get
flutter run
```

## گرفتن خروجی APK با GitHub Actions

اگر GitHub پوشه مخفی `.github` را از کامپیوتر آپلود نکرد، فقط این‌ها را آپلود کن:

```text
mobile
wordpress-plugin
README.md
GITHUB_ACTION_CODE.txt
```

بعد داخل خود GitHub این فایل را دستی بساز:

```text
.github/workflows/build-apk.yml
```

و کد داخل فایل `GITHUB_ACTION_CODE.txt` را داخلش کپی کن.

بعد برو تب Actions و workflow با نام `Build Android APK` را اجرا کن.

بعد از سبز شدن، پایین صفحه قسمت Artifacts فایل زیر را دانلود کن:

```text
banomi-manager-signed-apk
```

## ورود با اثر انگشت چطور کار می‌کند؟

1. بار اول باید آدرس سایت، نام کاربری و رمز را وارد کنی.
2. بعد از ورود موفق، اطلاعات در حافظه امن گوشی ذخیره می‌شود.
3. دفعه بعد صفحه ورود باز می‌شود و فیلدها پر هستند.
4. اگر روی گوشی اثر انگشت یا قفل امن فعال باشد، دکمه «ورود با اثر انگشت» نمایش داده می‌شود.
5. با تأیید اثر انگشت وارد برنامه می‌شوی.

## مشکلات رایج

### اکشن اجرا نمی‌شود

مسیر فایل باید دقیقاً این باشد:

```text
.github/workflows/build-apk.yml
```

اگر فایل کنار README باشد، اجرا نمی‌شود.

### APK نصب نمی‌شود

اگر نسخه قبلی اپ روی گوشی نصب است، حذفش کن و دوباره نصب کن.

### دکمه اثر انگشت کار نمی‌کند

روی گوشی باید اثر انگشت، Face Unlock یا رمز/PIN صفحه قفل فعال باشد. اگر هیچ قفل امنی فعال نباشد، اندروید اجازه ورود بیومتریک نمی‌دهد.

### دسته‌بندی یا تصویر نمی‌آید

افزونه جدید را نصب و فعال کن. نسخه قبلی افزونه تصویر و دسته‌بندی را کامل برنمی‌گرداند.

### خطای ورود

نام کاربری و رمز وردپرس را بررسی کن و مطمئن شو کاربر دسترسی مدیریت محصولات دارد.
