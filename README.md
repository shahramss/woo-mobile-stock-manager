# مدیریت سریع

اپلیکیشن Flutter فارسی برای مدیریت سریع محصولات ووکامرس و ارسال محصول به کانال بله.

- طراحی و ساخت: شهرام سعیدنیا
- پروژه: آقای رنجبر و جناب سجادی

## امکانات

- ورود با نام کاربری و رمز وردپرس
- ذخیره آدرس سایت، نام کاربری و رمز در حافظه امن گوشی
- ورود با اثر انگشت/قفل گوشی
- نمایش دسته‌بندی محصولات
- نمایش محصولات هر دسته همراه تصویر
- جستجو داخل همان دسته‌بندی
- ویرایش قیمت، تعداد موجودی و وضعیت موجود/ناموجود
- اگر موجودی را خالی بگذاری یا ۰ بزنی، یعنی محدودیت تعداد نداریم
- رنگ آبی برای محصول بروزرسانی‌شده تا ۲۴ ساعت
- رنگ سبز برای محصول ارسال‌شده به بله تا ۲۴ ساعت
- گرادیانت سبز/آبی وقتی محصول در ۲۴ ساعت اخیر هم بروزرسانی شده و هم به بله ارسال شده
- ارسال محصول به کانال بله به صورت پست تصویری
- امکان نوشتن متن دستی قبل از مشخصات و قیمت
- عدم نمایش موجودی در پست بله
- غیرفعال شدن ارسال دستی همان محصول تا یک ساعت
- ارسال خودکار محصولات یک دسته به بله با فاصله ۳ دقیقه، ۱ ساعت، ۷ ساعت یا ۲۴ ساعت


## رنگ‌بندی اکشن‌ها در لیست محصولات

```text
آبی: محصول در ۲۴ ساعت اخیر بروزرسانی شده است.
سبز: محصول در ۲۴ ساعت اخیر به بله ارسال شده است.
سبز/آبی گرادیانت: محصول در ۲۴ ساعت اخیر هم بروزرسانی شده و هم به بله ارسال شده است.
بعد از ۲۴ ساعت رنگ محصول به حالت عادی برمی‌گردد.
```

## موجودی بدون محدودیت

در صفحه جزئیات محصول، اگر فیلد «تعداد موجودی انبار» را خالی بگذاری یا مقدار ۰ وارد کنی، یعنی برای فروش/ارسال محدودیت تعداد نداریم. در این حالت مدیریت موجودی عددی ووکامرس برای آن محصول غیرفعال می‌شود، اما وضعیت موجود/ناموجود همچنان از سوییچ همان صفحه کنترل می‌شود.

## نصب افزونه وردپرس

داخل پوشه پروژه برو به:

```text
wordpress-plugin/woo-mobile-stock-manager-api
```

کل پوشه `woo-mobile-stock-manager-api` را zip کن.

بعد در وردپرس:

```text
پیشخوان وردپرس ← افزونه‌ها ← افزودن ← بارگذاری افزونه
```

فایل zip را نصب و فعال کن.

کاربر وردپرس باید یکی از این دسترسی‌ها را داشته باشد:

```text
manage_woocommerce
edit_products
```

## تنظیم بله

1. در بله با BotFather یک بازو بساز.
2. توکن بازو را بردار.
3. بازو را عضو/مدیر کانال کن.
4. حتماً اجازه ارسال پست در کانال داشته باشد.
5. داخل اپ، از صفحه دسته‌بندی روی آیکون تنظیمات بله بزن.
6. توکن بازو و نام کاربری کانال را وارد کن.

مثال شناسه کانال:

```text
@your_channel_username
```

توکن بله داخل اپ ذخیره نمی‌شود؛ در وردپرس ذخیره می‌شود.

## ارسال دستی محصول به کانال

1. وارد جزئیات محصول شو.
2. پایین صفحه بخش «ارسال در کانال بله» را پیدا کن.
3. متن دستی را وارد کن.
4. روی «ارسال در کانال» بزن.

پست شامل این موارد است:

- تصویر محصول
- متن دستی شما
- نام محصول
- مشخصات قابل نمایش محصول
- قیمت
- لینک محصول

موجودی انبار در پست نمایش داده نمی‌شود.

بعد از ارسال، دکمه همان محصول تا یک ساعت غیرفعال می‌شود.

## ارسال خودکار دسته‌بندی به کانال

1. وارد یک دسته‌بندی شو.
2. بالای لیست محصولات روی «ارسال خودکار» بزن.
3. فاصله ارسال را انتخاب کن:
   - هر ۳ دقیقه
   - هر ۱ ساعت
   - هر ۷ ساعت
   - هر ۲۴ ساعت
4. متن ثابت را بنویس.
5. روی «شروع ارسال» بزن.

هر بار یک محصول از همان دسته‌بندی ارسال می‌شود تا محصولات آن دسته تمام شود.

برای توقف، در همان دسته روی دکمه «توقف» بزن.

نکته مهم: ارسال خودکار با WP-Cron وردپرس انجام می‌شود. اگر سایت بازدید یا اجرای کرون نداشته باشد، ممکن است ارسال‌ها کمی دیرتر از زمان انتخاب‌شده انجام شوند.

## اجرای Flutter روی سیستم

```bash
cd mobile

flutter run
```

## ساخت APK با GitHub Actions

اگر پوشه `.github` را نمی‌توانی آپلود کنی، فقط این‌ها را آپلود کن:

```text
mobile
wordpress-plugin
README.md
GITHUB_ACTION_CODE.txt
```

بعد در GitHub بساز:

```text
.github/workflows/build-apk.yml
```

و محتوای `GITHUB_ACTION_CODE.txt` را داخلش کپی کن.

بعد برو به:

```text
Actions → Build Android APK
```

وقتی سبز شد، از بخش Artifacts فایل APK را دانلود کن.

## مشکلات رایج

### خطای ورود

- آدرس سایت را با `https://` وارد کن.
- افزونه وردپرس باید فعال باشد.
- کاربر باید اجازه مدیریت محصول داشته باشد.

### ارسال به بله انجام نمی‌شود

- توکن بازو را دوباره بررسی کن.
- شناسه کانال را با @ وارد کن.
- بازو باید مدیر کانال باشد.
- کانال باید اجازه ارسال پیام برای بازو داشته باشد.
- تصویر محصول باید از بیرون قابل دسترسی باشد.

### ارسال خودکار دقیقاً سر زمان انجام نمی‌شود

وردپرس از WP-Cron استفاده می‌کند و وابسته به اجرای سایت است. برای زمان‌بندی دقیق‌تر، روی هاست Cron واقعی تنظیم کن تا فایل `wp-cron.php` مرتب اجرا شود.

## تغییر نسخه ۱.۳.۰ - دکمه سفارش محصول در بله

در پست‌های ارسالی به کانال بله، لینک محصول دیگر به صورت متن ساده داخل کپشن نمایش داده نمی‌شود. زیر پست یک دکمه شیشه‌ای/اینلاین با متن زیر نمایش داده می‌شود:

`🛒 سفارش محصول`

این دکمه کاربر را مستقیم به لینک همان محصول در سایت هدایت می‌کند. برای کارکرد درست، محصول باید در ووکامرس لینک عمومی داشته باشد و بازوی بله باید در کانال ادمین باشد.

## تغییر نسخه ۱.۴.۰ - تغییر تصویر شاخص محصول

در صفحه جزئیات محصول، دکمه «تغییر تصویر شاخص» اضافه شد.

روش استفاده:

1. وارد جزئیات محصول شو.
2. بالای صفحه روی «تغییر تصویر شاخص» بزن.
3. تصویر جدید را از گالری گوشی انتخاب کن.
4. تصویر در وردپرس آپلود می‌شود و به عنوان تصویر شاخص همان محصول ذخیره می‌شود.

نکته‌ها:

- فرمت‌های مجاز: JPG، PNG، WEBP، GIF
- حداکثر حجم پیشنهادی: ۱۰ مگابایت
- کاربر وردپرس باید اجازه مدیریت/ویرایش محصول داشته باشد.
- بعد از تغییر تصویر، تصویر جدید در لیست محصولات و پست‌های بله استفاده می‌شود.

## تغییر نسخه ۱.۵.۰ - مرتب‌سازی و رنگ آخرین اکشن

در صفحه لیست محصولات، گزینه مرتب‌سازی اضافه شد:

- جدیدترین محصولات به قدیمی‌ترین
- قدیمی‌ترین محصولات به جدیدترین
- بیشترین قیمت به کمتر
- کمترین قیمت به بیشتر

رنگ کارت محصول بر اساس آخرین اکشن تغییر می‌کند:

- آبی: آخرین اکشن روی محصول، بروزرسانی قیمت/موجودی/تصویر بوده است.
- سبز: آخرین اکشن روی محصول، ارسال به کانال بله بوده است.

این وضعیت از سمت افزونه وردپرس ذخیره می‌شود و بعد از refresh یا ورود دوباره هم باقی می‌ماند.


## تغییر نسخه v9

- رنگ آبی محصول بروزرسانی‌شده و رنگ سبز محصول ارسال‌شده به بله فقط تا ۲۴ ساعت بعد از آخرین اکشن نمایش داده می‌شود.
- بعد از ۲۴ ساعت، محصول دوباره با رنگ عادی سفید در لیست نمایش داده می‌شود.

## تغییر نسخه ۱.۶.۰ - تم صورتی سازمانی

رنگ کلی اپلیکیشن بر اساس رنگ سازمانی صورتی بروزرسانی شد:

- هدرها و کارت‌های اصلی با گرادیانت صورتی کم‌رنگ و جذاب
- دکمه‌های اصلی با رنگ صورتی
- رنگ فوکوس فیلدها و آیکون‌های عمومی صورتی
- پس‌زمینه کلی اپ صورتی بسیار ملایم

نکته: رنگ‌های اکشن محصولات طبق درخواست قبلی حفظ شده‌اند؛ یعنی آبی برای بروزرسانی، سبز برای ارسال به بله و گرادیانت سبز/آبی برای هر دو اکشن.
