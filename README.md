# مدیریت سریع v16

نسخه اصلاح‌شده اپ مدیریت سریع برای ووکامرس.

## تغییرات v16

- رفع باگ ذخیره نشدن قیمت، موجودی، وضعیت موجودی و تصویر شاخص.
- تصویر شاخص اکنون بعد از انتخاب، با زدن دکمه «ذخیره تغییرات» روی سایت ذخیره می‌شود.
- بعد از ذخیره تغییرات، اپ محصول را دوباره از سرور می‌گیرد تا اطلاعات کش نشود.
- لینک تصویرها با کش‌شکن دریافت می‌شود تا تصویر قدیمی نمایش داده نشود.
- امکان نمایش گالری تصاویر محصول اضافه شد.
- امکان افزودن تصویر به گالری محصول اضافه شد.
- امکان حذف تصویر از گالری محصول با دکمه ضربدر کوچک روی عکس اضافه شد.
- تصاویر گالری در ابعاد 100×100 و به صورت اسکرول افقی نمایش داده می‌شوند.

## نصب افزونه

فایل افزونه داخل پوشه `wordpress-plugin` قرار دارد. برای نصب:

1. وارد پیشخوان وردپرس شوید.
2. افزونه قبلی مدیریت سریع را غیرفعال و حذف کنید.
3. فایل ZIP افزونه v16 را نصب کنید.
4. افزونه را فعال کنید.

## خروجی APK با GitHub Actions

اگر پوشه `.github` در ویندوز نمایش داده نشد، فایل `GITHUB_ACTION_CODE.txt` را باز کنید و محتوای آن را در مسیر زیر در GitHub بسازید:

```text
.github/workflows/build-apk.yml
```

بعد از push روی branch `main`، خروجی APK در بخش Actions > Artifacts ساخته می‌شود.

## نکته مهم

برای کارکرد گالری تصاویر و ذخیره مستقل تصویر شاخص، نصب افزونه v16 الزامی است. اگر اپ جدید نصب شود ولی افزونه قدیمی روی سایت بماند، امکانات جدید کار نمی‌کند.
