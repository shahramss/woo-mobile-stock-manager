# مدیریت سریع v19

نسخه اصلاح‌شده اپ مدیریت سریع برای ووکامرس.

## تغییرات v19

- رفع باگ بروزرسانی نشدن زمان نسبی سفارش‌ها.
- زمان سفارش‌ها دیگر فقط از متن تاریخ خوانده نمی‌شود؛ افزونه timestamp واقعی سفارش را هم می‌فرستد.
- با دکمه «بروزرسانی سفارش‌ها»، اطلاعات سفارش‌ها دوباره از سرور دریافت می‌شود و زمان‌ها مجدد محاسبه می‌شوند.
- هر دقیقه داخل خود اپ زمان‌ها دوباره محاسبه می‌شوند؛ مثلاً «۱۶ ساعت قبل» بعد از گذشت زمان به «۱۷ ساعت قبل» تغییر می‌کند.
- برای جلوگیری از کش، endpoint سفارش‌ها با header بدون کش و پارامتر no-cache کار می‌کند.

## امکانات نسخه‌های قبلی که حفظ شده‌اند

- مدیریت قیمت، موجودی، وضعیت موجودی، تصویر شاخص و گالری تصاویر محصول.
- پاک‌سازی کش LiteSpeed همان صفحه محصول بعد از بروزرسانی محصول.
- ارسال محصول به کانال بله با دکمه سفارش محصول.
- بخش سفارشات، جزئیات سفارش و ساخت فاکتور PNG.
- تم صورتی و آیکون صورتی.

## نصب افزونه

1. وارد پیشخوان وردپرس شوید.
2. افزونه قبلی مدیریت سریع را غیرفعال و حذف کنید.
3. فایل ZIP افزونه v19 را نصب کنید.
4. افزونه را فعال کنید.

## خروجی APK با GitHub Actions

اگر پوشه `.github` در ویندوز نمایش داده نشد، فایل `GITHUB_ACTION_CODE.txt` را باز کنید و محتوای آن را در مسیر زیر در GitHub بسازید:

```text
.github/workflows/build-apk.yml
```

بعد از push روی branch `main`، خروجی APK در بخش Actions > Artifacts ساخته می‌شود.

## نکته مهم

برای رفع کامل مشکل زمان سفارش‌ها، هم باید APK جدید v19 نصب شود و هم افزونه v19 روی وردپرس فعال باشد.
