# مدیریت سریع - نسخه v15

این نسخه شامل اصلاحات زیر است:

- تغییر تصویر شاخص محصول به صورت مستقل انجام می‌شود؛ لازم نیست قیمت یا موجودی را هم تغییر بدهید.
- بعد از انتخاب تصویر شاخص، اپ محصول را دوباره از سایت می‌گیرد و تصویر را با کش‌شکن نمایش می‌دهد.
- بخش سفارشات واضح‌تر شد و دکمه بزرگ «سفارشات فروشگاه» در صفحه اصلی اضافه شد.
- در صفحه سفارشات، پایین صفحه یک دکمه بزرگ «بروزرسانی سفارش‌ها» اضافه شد.
- با زدن بروزرسانی، سفارش‌ها، زمان ثبت سفارش و سفارش‌های جدید دوباره از سایت دریافت می‌شوند.
- در جزئیات سفارش، دکمه «دانلود فاکتور در گالری» اضافه شد.
- دکمه «ارسال یا پرینت فاکتور PNG» هم باقی مانده است.

## نصب افزونه

پوشه افزونه داخل مسیر زیر است:

`wordpress-plugin/woo-mobile-stock-manager-api`

برای نصب راحت، فایل ZIP جداگانه افزونه را در وردپرس نصب کنید.

## خروجی APK در GitHub

اگر پوشه `.github` در ویندوز دیده نشد، فایل `GITHUB_ACTION_CODE.txt` را باز کنید و محتوای آن را داخل این مسیر در GitHub بسازید:

`.github/workflows/build-apk.yml`

بعد از Push روی branch `main` خروجی APK در بخش Actions ساخته می‌شود.
