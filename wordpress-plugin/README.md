# Modiriat Sari API

این پوشه افزونه وردپرس اپ «مدیریت سریع» است.

برای نصب، پوشه `woo-mobile-stock-manager-api` را zip کنید و در وردپرس از بخش افزونه‌ها نصب کنید.

نسخه ۱.۵.۰:
- مرتب‌سازی محصولات بر اساس جدیدترین، قدیمی‌ترین، بیشترین قیمت و کمترین قیمت
- ذخیره آخرین اکشن محصول: بروزرسانی یا ارسال به بله
- ارسال مقدار `last_action` به اپ برای رنگی شدن کارت محصولات
