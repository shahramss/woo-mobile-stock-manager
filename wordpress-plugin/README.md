# Modiriat Sari API v16

افزونه اختصاصی اپ مدیریت سریع برای ووکامرس.

## امکانات

- ورود امن با یوزرنیم و پسورد وردپرس
- مدیریت قیمت، موجودی، وضعیت موجودی
- تغییر مستقل تصویر شاخص
- نمایش، افزودن و حذف تصاویر گالری محصول
- سفارشات ووکامرس
- ارسال محصول به کانال بله
- ارسال خودکار دسته‌بندی به بله

## مسیر API

```text
/wp-json/wmsm/v1
```

## نکته

کاربر وردپرس باید مجوز `manage_woocommerce` یا `edit_products` داشته باشد.
